# asm-fizzbuzz
fizzbuzz written in x86_64 assembly, for gnu/linux. probably also works on
other mostly-posix-compliant-unices.

## usage
```
fizzbuzz <arg>
```
arg is the upper bound and is expected to be a number in decimal system. it is
also necessary for the program to do anything.

## installation
not really sure why you would need it, but here you go:
```
make all
```
and then `bin` directory will appear with an executable. if for some reason you 
wanted to have it in your path, then easiest way would be to link to it in
`/usr/bin`.
